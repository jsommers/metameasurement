Artificial network load experiment

To generate the data, the Benchmarks/loadtests.sh was run, with “none” (to generate baseline data) and with no explicit extra load (but changing the output file names). The network load for this experiment was run using xfer.sh script (which creates a file of zeroes of some size and uploads/downloads using scp every 10 sec.

To plot the data, interfereplots.sh, which in turn calls interfere_plot.py, is run. Note that the command line has each name of the input data files.
